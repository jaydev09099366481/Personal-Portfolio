# 📖 About

This repository contains the source code for my personal portfolio website.

The portfolio was built to showcase my technical skills, academic background, featured projects, and professional experience as a Computer Science student and aspiring Software Developer. It serves as both an online resume and a central place where recruiters, clients, and collaborators can learn more about me and view my work.

The website focuses on:

- Modern and clean UI design
- Responsive layouts for desktop, tablet, and mobile devices
- Smooth scrolling animations
- Interactive project showcase
- Contact form integration
- Resume viewing and downloading
- Easy customization and maintenance

---

# ✨ Features

## 🏠 Home

- Professional landing page
- Animated hero section
- Brief introduction
- Call-to-action buttons

## 👤 About Me

- Personal introduction
- Educational background
- Career objectives
- Professional summary

## 💻 Skills

Categorized technical skills including:

- Frontend Development
- Backend Development
- Database Management
- Programming Languages
- Tools & Technologies

## 📂 Projects

Displays selected personal and academic projects with:

- Project title
- Description
- Technologies used
- Live Demo (if available)
- GitHub Repository

## 📄 Resume

Provides quick access to:

- View Resume
- Download Resume

## 📞 Contact

Interactive contact form allowing visitors to:

- Enter their name
- Email address
- Project type
- Message

The form automatically opens the visitor's default email client using a mailto link.

## ✨ Animations

- Scroll reveal effects
- Smooth transitions
- Hover animations
- Responsive navigation

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/jaydev09099366481/Personal-Portfolio.git
```

Navigate into the project folder:

```bash
cd Personal-Portfolio
```

---

## Run Locally

Simply open:

```text
index.html
```

in your preferred web browser.

You may also use the Visual Studio Code Live Server extension for automatic page reloading during development.

# 🌍 Deployment

The portfolio can be deployed easily using:

- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting

For GitHub Pages:

1. Push your project to GitHub.
2. Go to **Repository Settings**.
3. Select **Pages**.
4. Choose the **main** branch.
5. Save the settings.
6. Your portfolio will be published automatically.
